# Iconix

Lucide-style icon browser for the SVG collection in `icons/`.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## What it generates

- `generated/icon-manifest.json`: metadata used by the browser
- `generated/components/*.tsx`: standalone React icon components you can copy into other projects

The browser supports search, category/style filtering, SVG copy, SVG download, and copying a ready-to-paste React component source file for any icon.
